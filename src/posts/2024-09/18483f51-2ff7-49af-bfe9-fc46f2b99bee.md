### 摘要

使用TypeScript将字符串hash值转化为对应rgb颜色。


### 代码实现

```typescript
function hashStringToColor(str) {
  let hash = 0;
  for (let i = 0; i < str.length; i++) {
    hash = str.charCodeAt(i) + ((hash << 5) - hash);
  }
  const color = (hash & 0x00FFFFFF)
    .toString(16)
    .toUpperCase()
    .padStart(6, `0`);
  return `#${color}`;
}
```
