### 摘要

本文提供了一种在<a>PC</a>网页端中修改<a>哔哩哔哩</a>任意倍速播放视频的方法:


### 方法

```javascript
// 打开浏览器开发者工具, 默认快捷键 `ctrl + shift + i` 或者 `F12`
// 输入以下命令并执行
document.querySelector("video").playbackRate = n;
```

::: info
<a>`n`</a>代表播放倍速, 可支持<a>浮点类型</a>
:::

::: warning
视频更换时需要重新设置播放倍速
:::
