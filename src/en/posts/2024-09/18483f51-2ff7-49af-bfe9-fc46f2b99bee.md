### Abstract

Convert a string's hash value into a corresponding RGB color using TypeScript.

### Coding

```typescript
function hashStringToColor(str) {
  let hash = 0;
  for (let i = 0; i < str.length; i++) {
    hash = str.charCodeAt(i) + ((hash << 5) - hash);
  }
  const color = (hash & 0x00FFFFFF)
    .toString(16)
    .toUpperCase()
    .padStart(6, `0`);
  return `#${color}`;
}

```
